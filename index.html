Les Normes de Calcul

Comprendre les différences fondamentales entre BAEL 91 et Eurocode 2

BAEL 91

Historique

Le Béton Armé aux États Limites (BAEL) 91 est la norme historique française utilisée jusqu'en 2010.

Principales caractéristiques

  • Approche semi-probabiliste
  • Coefficient de sécurité γb = 1.5 pour le béton
  • Coefficient de sécurité γs = 1.15 pour l'acier
  • Contrainte admissible du béton : fbu = 0.85fc28/(γbθ)
  • Diagramme rectangulaire simplifié

Domaine d'application

Le BAEL 91 s'applique aux structures en béton armé courantes : bâtiments, ponts, réservoirs, etc.

Vérifications principales

  • Équilibre des sections
  • Non-fragilité
  • Flèches admissibles
  • Fissuration (peu/préjudiciable/très préjudiciable)

Eurocode 2

Européen

L'Eurocode 2 (EN 1992) est la norme européenne actuelle pour le calcul des structures en béton.

Principales caractéristiques

  • Approche performantielle
  • Coefficient de sécurité γc = 1.5 pour le béton
  • Coefficient de sécurité γs = 1.15 pour l'acier
  • Contrainte admissible : fcd = αccfck/γc
  • Diagramme parabole-rectangle ou rectangulaire

Domaine d'application

L'Eurocode 2 couvre un large éventail de structures : bâtiments, ponts, silos, réservoirs, tours, etc.

Vérifications principales

  • États limites ultimes (ELU)
  • États limites de service (ELS)
  • Durabilité et enrobages
  • Maitrise de la fissuration
  • Analyse non-linéaire si nécessaire

Paramètres de calcul

Recommandations BAEL 91

  • Vérifier le moment réduit μ ≤ μlim = 0.371
  • Armature minimale : As,min = 0.23bdft28/fe
  • Vérifier la contrainte de compression du béton
  • Vérifier les flèches si nécessaire

Recommandations Eurocode 2

  • Vérifier le moment réduit μ ≤ μlim = 0.371
  • Armature minimale : As,min = 0.26fctmbtd/fyk
  • Vérifier l'ouverture des fissures en ELS
  • Vérifier les flèches avec la méthode simplifiée

Paramètres de calcul

Recommandations BAEL 91

  • Calculer l'élancement λ = L0/i
  • Déterminer le coefficient α = 0.85/(1+0.2(λ/35)²)
  • Utiliser la section réduite Br = (b-2)(h-2)
  • Vérifier λ ≤ 70
  • Armature minimale : 4 cm² ou 0.2% de la section

Recommandations Eurocode 2

  • Calculer l'élancement λ = L0/i
  • Déterminer le coefficient α = 0.86/(1+(λ/62)²)
  • Utiliser la section brute Ac = b×h
  • Vérifier λ ≤ 120
  • Armature minimale : 0.2% de la section

Paramètres de calcul

Recommandations BAEL 91

  • Vérifier la contrainte du sol : q = N/(B×L) ≤ qadmissible
  • Calculer le moment : M = q×s²/2 avec s = (B-c)/2
  • Vérifier le poinçonnement : VEd ≤ VRd
  • Armature minimale : 3.5 cm²/m
  • Enrobage minimal : 3 cm

Recommandations Eurocode 2

  • Vérifier la contrainte du sol : q = N/(B×L) ≤ qadmissible
  • Calculer le moment : M = q×s²/2 avec s = (B-c)/2
  • Vérifier le poinçonnement : vEd ≤ vRd,c
  • Armature minimale : As,min = 0.0013×b×d
  • Enrobage selon classe d'exposition

Avantages du BAEL 91

1
Simplicité
Règles plus simples et plus directes, faciles à appliquer manuellement
2
Conservatisme
Approche plus sécuritaire avec des coefficients plus élevés
3
Familiarité
Bien connu des ingénieurs français avec une longue pratique

Performance

3
Durabilité
Meilleure prise en compte de la durabilité et de l'environnement

Ressources et Formules

Formules clés et références pour approfondir vos connaissances

Poutres - Formules clés

BAEL 91

fbu = 0.85fc28/(γbθ)
μ = Mu/(b·d²·fbu)
α = 1.25(1-√(1-2μ))
z = d(1-0.4α)
As = Mu/(z·fyd)

Eurocode 2

fcd = αccfck/γc
μ = MEd/(b·d²·fcd)
α = 1.25(1-√(1-2μ))
z = d(1-0.4α)
As = MEd/(z·fyd)

Poteaux - Formules clés

BAEL 91

λ = L0/i
α = 0.85/(1+0.2(λ/35)²)
Br = (b-2)(h-2)
Nu ≤ α(Brfbu/0.9 + Asfyd)

Eurocode 2

λ = L0/i
α = 0.86/(1+(λ/62)²)
Ac = b·h
NEd ≤ α(Acfcd + Asfyd)

Semelles - Formules clés

BAEL 91 & Eurocode 2

q = Nu/(B·L)
sx = (L-c1)/2
sy = (B-c2)/2
Mx = q·sx²/2
My = q·sy²/2

Poinçonnement (EC2)

vEd = VEd/(u·d)
vRd,c = 0.18/γck(100ρfck)1/3
